Welcome to HSS Pages.

HSS is short for "HEI SHENG SHA", a card-playing game like "SAN GUO SHA". The differences between them is playing roles and many other aspects. Source code includes client and server code. Client code is based on PYTHON while server code based on erlang, a concurrency-oriented programming language. The most important thing for us is: HAVE FUN!!!

How to get source code, sorry, you can't checkin any code

$ cd your_repo_root/repo_name
$ git clone git@github.com:spookyy/hss.git

Why python for client

Uhh, the truth is no reason. I myself very like python. That's the only reason. Long long ago when I tried to find a language for client part, I thought why not, python can do everything for me. Another reason is that I'm a newbie, I need practice. As the old saying goes, practice makes perfect.

To run this code, you should have python, pyQt4, cocos2d-python, pygame in your enviroment.

FYI:

 loginpage.py ---> login page, you should firstly and only run it
 hall.py      ---> hall page, you can either create a room or 
                   choose a room to join
 room.py      ---> the "main" page where you actually playing game

how to run the client

$ python loginpage.py

Why erlang for server

Interesting. Erlang is a programming language used to build massively scalable soft real-time systems with requirements on high availability. Some of its uses are in telecoms, banking, e-commerce, computer telephony and instant messaging. Erlang's runtime system has built-in support for concurrency, distribution and fault tolerance. I am familiar with it and it is convenient and quick to build a realiable server with erlang. YOu can get anything about erlang at:http://www.erlang.org/. How to start server

To build this code, run the following command:

1) to build src files

$ erlc -o ebin src/*.erl

2) start erl shell

$ erl -pa ebin/ -sname unode@uhost -mnesia dir '"/where/your/hss/is/"'

3) You should start the mnesia in the shell

$erl> mnesia:start().

4) In erl shell, you should run:

$erl> application:start(hss).
